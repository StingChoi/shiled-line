# 유튜브 자동 업로드 - 최초 1회 설정

> 딱 한 번만 하면 됩니다. 그 뒤로는 배치파일이 알아서 올립니다.

---

## 1단계: YouTube Data API 켜기 (브라우저)

1. https://console.cloud.google.com 접속 (블로거 설정할 때 쓴 그 구글 계정).
2. 화면 위쪽에서 프로젝트가 **`second-raceway-494802-p5`** 인지 확인.
3. 왼쪽 메뉴 → **API 및 서비스 → 라이브러리**.
4. 검색창에 **YouTube Data API v3** 입력 → 클릭 → **사용 설정(Enable)** 버튼.

> 블로거 발행과 같은 프로젝트라, 이 API만 켜면 됩니다.

---

## 2단계: 첫 업로드 때 로그인 (1회)

1. 명령창에서 한 번 실행:
   ```
   cd /d D:\shiled-line
   py upload_videos.py
   ```
   (또는 그냥 `홈페이지+블로거_한방발행.bat` 더블클릭 — [1/4]에서 자동 실행됨)
2. 브라우저가 열리면 **유튜브 채널 계정**으로 로그인.
3. 혹시 **"Google에서 확인하지 않은 앱"** 경고가 뜨면:
   - **고급** 클릭 → **"(앱 이름)(으)로 이동(안전하지 않음)"** 클릭.
   - (내가 직접 만든 내 앱이라 괜찮습니다. 외부에 공개된 앱이 아닙니다.)
4. 권한 요청 화면에서 **허용**.
5. 끝나면 `token_youtube.json` 파일이 자동으로 생깁니다. 이후로는 로그인 안 해도 됩니다.

---

## 참고

- **업로드 공개 범위**: 현재 **공개(public)** 로 설정돼 있습니다.
  - 미등록으로 바꾸려면 `upload_videos.py` 안의 `PRIVACY = "public"` 을 `"unlisted"` 로 수정.
- **하루 업로드 한도**: 유튜브 기본 할당량으로 하루 약 6개까지 올릴 수 있습니다. 밀린 작업을 한 번에 너무 많이 올리지만 않으면 충분합니다.
- **보안**: `token_youtube.json` 은 인증 열쇠라 GitHub에 안 올라갑니다(.gitignore 차단됨). 절대 남에게 주지 마세요.
- 문제가 생기면 명령창에 뜬 영문 오류 메시지를 코워크에게 그대로 보여주세요.
